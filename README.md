# 🌙 MOONFLIX - Open Source Entertainment Service

Moonflix is a lightweight, high-performance, and open-source streaming interface built with HTML, CSS, and JavaScript. It uses the TMDB API to fetch movie and TV show data and provides a premium, Netflix-inspired user experience.

## 🚀 Features
- **Cinematic UI:** Large hero banner and responsive movie grid.
- **Detailed Previews:** Interactive modal with movie descriptions, ratings, and backdrops.
- **Built-in Player:** Seamlessly watch content with an overlay player.
- **Dynamic Search:** Real-time search functionality for the entire TMDB database.
- **Open Source:** Designed to be easily forked and customized.

## 🛠️ Getting Started

### 1. Obtain an API Key
To use this service, you need a free API Key from [The Movie Database (TMDB)](https://www.themoviedb.org/).
- Register/Login to TMDB.
- Go to **Settings -> API**.
- Create a "Developer" key.

### 2. Setup
1. Fork or download this repository.
2. Open `index.html`.
3. Locate the line: `const API_KEY = 'YOUR_API_KEY_HERE';`
4. Replace it with your actual API key.
5. Host the file using **GitHub Pages**, **Cloudflare Pages**, or simply open it in your browser.
